# موقع بطاقة تهنئة عيد الفطر - Perfection

موقع بسيط وأنيق لإنشاء بطاقات تهنئة عيد الفطر مخصصة.

## المميزات

- إضافة الاسم على البطاقة بخط Tajawal BOLD
- تحميل البطاقة كصورة
- مشاركة البطاقة على مواقع التواصل الاجتماعي
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة عربية كاملة

## الملفات المطلوبة

- `index.html` - الصفحة الرئيسية
- `eid_elfitr_poster.jpeg` - صورة البوستر
- `icon.png` - أيقونة الموقع (favicon)
- `vercel.json` - ملف إعدادات Vercel

## التشغيل المحلي (مهم)

لتجنب مشاكل `file://` في تحميل أو مشاركة الصورة، شغل الموقع من سيرفر محلي:

```bash
npx serve .
```

ثم افتح الرابط الذي يظهر لك (مثل `http://localhost:3000`).

## الرفع على Vercel

### الطريقة الأولى: عبر Vercel CLI

1. قم بتثبيت Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. من داخل مجلد المشروع، قم بتشغيل:
   ```bash
   vercel
   ```

3. اتبع التعليمات لربط المشروع بحسابك على Vercel

### الطريقة الثانية: عبر موقع Vercel

1. قم بإنشاء حساب على [vercel.com](https://vercel.com)
2. اضغط على "New Project"
3. ارفع المجلد الذي يحتوي على الملفات
4. سيتم نشر الموقع تلقائياً
